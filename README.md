# VOIDFILL.fm

Live underground radio station with an AI DJ. Built as a single HTML file — no framework, no backend, no dependencies.

**→ [haulstreet.github.io/voidfill-fm](https://haulstreet.github.io/voidfill-fm)**

---

## Stations

Five live streams, free and ad-free 24/7:

| Station | Vibe |
|---|---|
Bossa Nova · Samba · Brazilian Soul |
Vintage Ambient · Downtempo · IDM |
Easy Listening |
Liquid Funk · Jazz · Soulful Electronic |
Indie Pop · Electropop · Alternative |

## AI Features

Powered by **Groq** (free tier — no credit card needed):

- **Song suggestions** — DJ FREQ analyses the station you're on and recommends 7 real tracks to discover, with reasons
- **DJ chat** — talk to FREQ about music, artists, or what to listen to next

## Stack

HTML · CSS · Vanilla JS · [Groq API](https://console.groq.com) · [SomaFM](https://somafm.com)

## Setup

1. Get a free API key at [console.groq.com](https://console.groq.com)
2. Open `index.html` and replace `YOUR_GROQ_API_KEY_HERE` with your key
3. Done

## Deploy

Hosted on GitHub Pages. Any edit via [github.dev](https://github.dev) (press `.` on the repo) → commit → live in ~60 seconds.
